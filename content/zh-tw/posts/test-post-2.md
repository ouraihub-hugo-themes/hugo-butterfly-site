# Tailwind CSS 实践

Tailwind CSS 是一個功能類優先的 CSS 框架。

## 核心概念

- 实用工具類
- 响應式設計
- 暗色模式
- 自定义配置

## 基本用法

```html
<div class="bg-blue-500 text-white p-4 rounded-lg">
  Hello Tailwind!
</div>
```

## 响應式設計

```html
<div class="w-full md:w-1/2 lg:w-1/3">
  响應式容器
</div>
```

## 总结

Tailwind CSS 让樣式開發变得更加高效。
