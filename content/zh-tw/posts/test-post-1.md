# Hugo 入門指南

Hugo 是一個快速、現代的静態網站生成器。

## 為什么選擇 Hugo

- 极快的构建速度
- 简单易用
- 功能強大
- 活跃的社区

## 安裝 Hugo

```bash
# macOS
brew install hugo

# Windows
choco install hugo-extended

# Linux
snap install hugo
```

## 創建新站點

```bash
hugo new site my-site
cd my-site
```

## 总结

Hugo 是构建静態網站的绝佳選擇。
