# Hugo 入门指南

Hugo 是一个快速、现代的静态网站生成器。

## 为什么选择 Hugo

- 极快的构建速度
- 简单易用
- 功能强大
- 活跃的社区

## 安装 Hugo

```bash
# macOS
brew install hugo

# Windows
choco install hugo-extended

# Linux
snap install hugo
```

## 创建新站点

```bash
hugo new site my-site
cd my-site
```

## 总结

Hugo 是构建静态网站的绝佳选择。
