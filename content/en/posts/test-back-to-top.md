# Back to Top Functionality Test

This is a long article for testing the back to top functionality. When you scroll down more than 300px, you should see a back to top button in the bottom right corner.

## Features

1. **Scroll Monitoring** - Button appears when scrolling exceeds 300px
2. **Smooth Scrolling** - Clicking the button smoothly scrolls to the top
3. **Scroll Percentage** - Displays current scroll percentage
4. **Fade Animation** - Smooth animation when button appears/disappears
5. **Responsive Design** - Good display on both mobile and desktop
