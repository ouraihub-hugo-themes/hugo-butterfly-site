# Hugo Getting Started Guide

Hugo is a fast, modern static site generator.

## Why Choose Hugo

- Extremely fast build speed
- Easy to use
- Powerful features
- Active community

## Installing Hugo

```bash
# macOS
brew install hugo

# Windows
choco install hugo-extended

# Linux
snap install hugo
```

## Creating a New Site

```bash
hugo new site my-site
cd my-site
```

## Summary

Hugo is an excellent choice for building static websites.
